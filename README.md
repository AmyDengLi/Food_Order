# pizza.app
使用vue框架实现的点餐系统，通过野狗平台创建的一个小小的数据库，帮助实现后台的存储/读取数据，借助bootstrap布局系统页面，实现的功能有：
1、提供外卖食物统计、商家的基本信息；
2、实现购物车的基本功能；
3、用户可以自行添加或者删除食物列表；
4、实现用户的登录和注册
